# Money Conversion App

A secure platform for users under 18 to convert and transfer money to their bank accounts with a free 2-month trial period.

## Features

- **Age-Verified Accounts**: Secure registration for users 17 and younger
- **Free Trial Period**: 2 months of free service to explore the platform
- **Flexible Subscription**: Optional paid subscription after trial ($X.XX/month)
- **Cancel Anytime**: Users can cancel their subscription at any time
- **Secure Bank Integration**: Direct money conversion and transfer to bank accounts
- **Transaction History**: Track all conversions and transfers
- **Real-time Balance**: Check your account balance anytime

## Tech Stack

- **Frontend**: React.js with TypeScript
- **Backend**: Node.js with Express.js
- **Database**: MongoDB
- **Authentication**: JWT with Age Verification
- **Payment Processing**: Stripe API
- **Banking Integration**: Plaid API (for bank connections)
